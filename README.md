u14rubpls
================

Shippable CI image for Ruby on Ubuntu 14.04 with commonly used services. Available versions are:

1. 1.8.7
2. 1.9.3
3. 2.0.0
4. 2.1.5
5. 2.2.1

**Services:**

1. elasticsearch 1.5
2. memcached 1.4
3. mongodb 3.0
4. mysql 5.6
5. postgres 9.4
6. rabbitmq 3.5
7. redis 3.0
8. selenium 2.45
9. sqllite 3
